Word2vec usando ingredientes.

Todas las recetas tienen una serie de ingredientes, que se usan en este caso como un "documento" del cual se extraen las "palabras". Con esto, cada ingrediente va a tener una representación única, un vector de 100 dimensiones, con el que se puede operar.
